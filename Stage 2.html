Stage 0: The Basics of Web and HTML - Getting Started with HTML

Intro to the Web
  • WWW - The World Wide Web is the network of networks. It provides the paths to connect to content anywhere in the world.
  • HTML - Hypertext Markup Language is the language used to define the structure and element information of a webpage. The text defines what you see and the markup defines how it looks in the browser.
  • Images, videos and music are referenced and linked to in the HTML document.
  • Browser - website viewer
  • HTTP - Hypertext Transfer Protocol is the application protocol of the web. It decodes HTML into a readable form for humans.
  • Internet - The Internet is the largest computer network in the world and is comprised of all public computers and servers.
  • Servers - Servers are specialized computers that host files used for websites and other accessable contents for use on the internet.
HTML Markup and Tags
  • Opening <tag>
  • Contents - text between the tags
  • Closing Tag - </tag>
  • Element - opening, contents and closing together
  • b - bold
  • em - emphisis italics
  • a - anchor (a href="url">contents/a)
  • (img src="url" alt="text")
  • alternate text - displayes text if there is a broken request
  • br - line break
  • p - paragraph
Elements

Block elements boxify the content contained inside the block element tag. Below are some examples of both inline and block tags.

  • inline - a, br, img, span, strong
  • block - div, p - paragraph, form - height and width can be modified

Stage 2.1: Introduction to "Serious" Programming

Computer Programing

Machines are usually designed to perform a specific task or a few similar tasks. Computers on the other hand can be programmed to do whatever your mind can think of. We as programmers can break the task down into a specific sequence of instructions for the computer to process.

Programming Languages

There are many different programming languages to choose from. They all have their pros and cons depending on what you are trying to accomplish. Python is an example of a programming language.

Programming Grammar

Each programming language has a defined grammar model applied in the language syntax. An Interpreter converts the Python code from the syntax humans can understand into a code that computers understand (probably binary) to execute the process. Programming grammar is very precise and details such as spaces and capital letters where they should not be can cause a syntax error and cause the program to fail. Your Python code must follow the syntax perfectly. The Python Interpreter will not interpret the code like a human would and be able to infer anything outside of its grammatical structure. Computers are not smart enough to make inferences and piece together improper syntax.

Python Expression

A Python Expression is a Python statement that uses perfectly applied syntax using the defined grammar to get the desired result.

Median

# Define a procedure, median, that takes three
# numbers as its inputs, and returns the median
# of the three numbers.
# Make sure your procedure has a return statement.

def bigger(a,b):
    if a > b:
        return a
    else:
        return b
def biggest(a,b,c):
    return bigger(a,bigger(b,c))
def median(a,b,c):
    big = biggest(a,b,c)
    if big == a:
        return bigger(b,c)
    if big == b:
        return bigger(a,c)
    else:
        return bigger(a,b)

print(median(1,2,3))
#>>> 2
print(median(9,3,6))
#>>> 6
print(median(7,8,7))
#>>> 7
